# Baby steps

## Syntax

Clojure has a very simple syntax because it is a Lisp.

### Values

    ;; integer
    1

    ;; String
    "hello"

    ;; Symbol
    'my-symbol

    ;; Keyword
    :my-keyword

    ;; Lists
    '(1 2 3 4 5)

    ;; Vector
    [1 2 3 4 5]

    ;; Assoc
    {:name "Endy" :car "Subaru Impreza" :number-of-pets 5}

    ;; Set
    #{1 2 3 4}

### Vars

    (def name "Endy")
    (def car "Subaru Impreza")
